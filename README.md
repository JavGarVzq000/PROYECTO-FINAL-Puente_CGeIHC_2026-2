# Proyecto Final - (Planta baja) Puente

- Proyecto para la clase de Computación Gráfica e Interacción Humano - Computadora.
- Recorrido virtual del puente (Planta baja), reconstruido con ayuda de modelos tridimensionales.
- Programado en C++.

## Características 
- Modelos hechos en 3ds Max.
- Texturas modificadas en GIMP.
- Código hecho en Visual Studio 2026.
- Modelo del puente (Planta baja).
- Modelos disponibles en: https://drive.google.com/drive/folders/1ILzTw2GE3oRpg-UWSyoVG6pnlWk1Tch8?usp=sharing
